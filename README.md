# laravel vue Metro UI CRUD
CRUD Application with laravel vue and Metro UI.
## Git Clone or download zip manualy
```git clone https://github.com/maheshkareeya/laravel-vue.git```
## Install Node dependencies 
```npm install```
## Start Vue 
```npm run serve```
## Open another tab 
## Change directory
```composer install```
### Start Laravel
```php artisan serve```
### Open following link
http://localhost:8080
